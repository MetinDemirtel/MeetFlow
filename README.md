Overview

MeetFlow is a comprehensive, offline-first meeting management application with AI-powered content analysis, multi-format document processing, and intelligent transcription capabilities. Designed for enterprise use, it provides real-time note-taking and automated content extraction while maintaining complete data privacy through 100% offline operation.

--------------------------------------------------------------------------------
Key Features
--------------------------------------------------------------------------------

Core Functionality:
- Real-time Note Taking with auto-save (2-second delay)
- Meeting Timer: Start/pause/reset with HH:MM:SS display
- Bilingual Support: Full TR/EN language switching
- Offline-First: All data stored locally in browser
- Export: Word (.doc) and TXT with Turkish character handling

Advanced Modules:

1. Notes Module
   - Primary note-taking interface
   - Auto-save with visual confirmation
   - Quick action tags (DECISION, ACTION, RISK, QUESTION)
   - Full keyboard support

2. Transcript Processing
   - Dual Input: Text mode (copy-paste) and Image mode (OCR screenshots)
   - Automatic speaker detection
   - Timestamp parsing
   - Direct integration with notes

3. Presentation Analysis
   - Formats: PNG, JPG, PDF, PPT, PPTX
   - Multi-page PDF support
   - OCR simulation with realistic content extraction
   - Batch processing
   - Notes integration

4. Participant Management
   - Upload participant list screenshots
   - Automatic name detection (OCR)
   - Alphabetical sorting (Turkish locale)
   - Duplicate prevention
   - One-click transfer to attendees field

5. Analytics Dashboard
   - Word count statistics
   - Action items, decisions, risks, questions tracking
   - Meeting duration display

6. User Guide
   - Feature documentation
   - Quick reference
   - Privacy information

--------------------------------------------------------------------------------
Privacy & Security
--------------------------------------------------------------------------------

- 100% offline operation
- Data stored in browser LocalStorage only
- No server transmission
- Complete data ownership by user

--------------------------------------------------------------------------------
Usage
--------------------------------------------------------------------------------

Starting a Meeting:
1. Enter meeting title
2. Add attendees (comma-separated)
3. Click "START" to begin timer
4. Take notes in Notes tab

Processing Transcripts:
1. Go to Transcript tab
2. Choose Text or Image mode
3. Paste or upload content
4. Click "PROCESS TRANSCRIPT"

Adding Presentations:
1. Go to Presentation tab
2. Upload files (PNG, JPG, PDF, PPT, PPTX)
3. Click "PROCESS ALL FILES"
4. Add extracted content to notes

Managing Participants:
1. Go to Participants tab
2. Upload screenshots
3. Click "GENERATE LIST"
4. Transfer to attendees field

Exporting:
1. Click "WORD" button
2. File downloads automatically
3. Format: Title_YYYY-MM-DD.doc

Limitations

- Storage limited by browser LocalStorage quota
- No cloud sync
- No real-time collaboration

Genel Bakış

MeetFlow, yapay zeka destekli içerik analizi, çoklu format belge işleme ve akıllı transkripsiyon yeteneklerine sahip, offline-öncelikli kapsamlı bir toplantı yönetim uygulamasıdır. Kurumsal kullanım için tasarlanmış olup, %100 çevrimdışı çalışarak tam veri gizliliği sağlarken gerçek zamanlı not alma ve otomatik içerik çıkarma sunar.

--------------------------------------------------------------------------------
Temel Özellikler
--------------------------------------------------------------------------------

Ana Fonksiyonlar:
- Otomatik kayıtlı gerçek zamanlı not alma (2 saniye gecikme)
- Toplantı Zamanlayıcısı: Başlat/duraklat/sıfırla (SS:DD:SS)
- Çift Dil Desteği: Tam TR/EN dil değiştirme
- Offline Öncelikli: Tüm veriler tarayıcıda yerel olarak saklanır
- Dışa Aktarma: Türkçe karakter desteğiyle Word (.doc) ve TXT

Gelişmiş Modüller:

1. Notlar Modülü
   - Ana not alma arayüzü
   - Görsel onaylı otomatik kayıt
   - Hızlı aksiyon etiketleri (KARAR, AKSİYON, RİSK, SORU)
   - Tam klavye desteği

2. Transkript İşleme
   - Çift Giriş: Metin modu (kopyala-yapıştır) ve Görüntü modu (OCR)
   - Otomatik konuşmacı tespiti
   - Zaman damgası ayrıştırma
   - Notlarla doğrudan entegrasyon

3. Sunum Analizi
   - Formatlar: PNG, JPG, PDF, PPT, PPTX
   - Çok sayfalı PDF desteği
   - Gerçekçi içerik çıkarmalı OCR simülasyonu
   - Toplu işleme
   - Not entegrasyonu

4. Katılımcı Yönetimi
   - Katılımcı listesi ekran görüntülerini yükleme
   - Otomatik isim tespiti (OCR)
   - Alfabetik sıralama (Türkçe yerel ayar)
   - Tekrar önleme
   - Katılımcılar alanına tek tıkla aktarma

5. Analiz Panosu
   - Kelime sayısı istatistikleri
   - Aksiyon, karar, risk, soru takibi
   - Toplantı süresi gösterimi

6. Kullanım Kılavuzu
   - Özellik dokümantasyonu
   - Hızlı referans
   - Gizlilik bilgileri

Kullanım
--------------------------------------------------------------------------------

Toplantı Başlatma:
1. Toplantı başlığı girin
2. Katılımcıları ekleyin (virgülle ayırarak)
3. Zamanlayıcıyı başlatmak için "BAŞLAT"a tıklayın
4. Notlar sekmesinde not alın

Transkript İşleme:
1. Transkript sekmesine gidin
2. Metin veya Görüntü modunu seçin
3. İçeriği yapıştırın veya yükleyin
4. "TRANSKRİPTİ İŞLE" butonuna tıklayın

Sunum Ekleme:
1. Sunum sekmesine gidin
2. Dosyaları yükleyin (PNG, JPG, PDF, PPT, PPTX)
3. "TÜM DOSYALARI İŞLE" butonuna tıklayın
4. Çıkarılan içeriği notlara ekleyin

Katılımcı Yönetimi:
1. Katılımcılar sekmesine gidin
2. Ekran görüntülerini yükleyin
3. "LİSTE OLUŞTUR" butonuna tıklayın
4. Katılımcılar alanına aktarın

Dışa Aktarma:
1. "WORD" butonuna tıklayın
2. Dosya otomatik indirilir
3. Format: Baslik_YYYY-MM-DD.doc
