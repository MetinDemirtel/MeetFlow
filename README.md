# MeetFlow | Smart Meeting Intelligence

[TR] MeetFlow, regülasyonlar nedeniyle bulut tabanlı AI araçlarının kısıtlandığı kurumsal ortamlar için geliştirilmiş, %100 yerel çalışan bir toplantı asistanıdır. Veri güvenliğinden ödün vermeden, ekran görüntüsü üzerinden OCR ile verimlilik sağlar.

[EN] MeetFlow is a 100% offline meeting assistant for regulated environments where cloud AI is restricted. It ensures secure productivity via local OCR without data leakage.

---

### Motivasyon / Motivation

TR: Kurumsal ağlarda ve bankacılık gibi sektörlerde veri güvenliği nedeniyle yapay zeka araçlarına erişim sınırlıdır. MeetFlow, veriyi dışarı çıkarmadan, ekran görüntülerini metne dönüştürerek bu engeli aşar. Teorideki teknolojiyi, kurumsal gerçeklerle buluşturan güvenli bir köprüdür.

EN: Access to AI is often restricted in regulated sectors due to security policies. MeetFlow bypasses this by using local OCR to process data from screenshots without leaving the local environment. It is a secure bridge between modern AI potential and corporate realities.

---

### Ozellikler / Features

- 100% Offline: Veri asla cihazınızdan çıkmaz. / Data never leaves your device.
- OCR Integration: Ekran görüntüsünden metin üretir. / Converts screenshots to editable text.
- Privacy First: Regülasyonlara tam uyumlu. / Fully compliant with data regulations.
- No Cloud: Sunucu veya API gerektirmez. / No servers or APIs required.
